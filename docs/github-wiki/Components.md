| `ENVICard` | `ENVICard.swift` | Elevated card container |
| `ENVIChip` | `ENVIChip.swift` | Filter / action chips |
| `ENVIInput` | `ENVIInput.swift` | Label + text/secure field + validation message |

| `ENVIEmptyState` | `ENVIEmptyState.swift` | Empty state placeholder |
| `ENVIErrorBanner` | `ENVIErrorBanner.swift` | Error notification banner |
| `ENVIFilterChip` | `ENVIFilterChip.swift` | Filterable chip variant |
| `ENVILoadingState` | `ENVILoadingState.swift` | Loading state indicator |
| `ENVIPlatformFilterBar` | `ENVIPlatformFilterBar.s...` | Platform filter horizontal bar |
