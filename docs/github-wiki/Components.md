# Components

**Last updated:** 2026-04-03 UTC

Reusable UI under `ENVI/Components/` (SwiftUI unless noted).

| Component | File | Role |
|-----------|------|------|
| `ENVIBadge` | `ENVIBadge.swift` | Status badges |
| `ENVIBottomSheet` | `ENVIBottomSheet.swift` | UIKit bottom sheet presentation |
| `ENVIButton` | `ENVIButton.swift` | Primary / secondary / ghost |
| `ENVICard` | `ENVICard.swift` | Elevated card container |
| `ENVIChip` | `ENVIChip.swift` | Filter / action chips |
| `ENVIInput` | `ENVIInput.swift` | Label + text/secure field + validation message |
| `ENVIProgressRing` | `ENVIProgressRing.swift` | Circular progress |
| `ENVITabBar` | `ENVITabBar.swift` | Custom floating tab bar (**UIKit**), used by `MainTabBarController` |
| `ENVIToggle` | `ENVIToggle.swift` | Custom toggle |

Feature-specific views (e.g. `KPICardView`, `ExpandableFeedCardView`) live under `Features/` — not listed here.
